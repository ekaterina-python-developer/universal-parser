# Универсальный парсер для сайтов.

Универсальный парсер для поиска контактных данных (Email и телефоны).

## Особенности решения
- **Движок:** Использование `Crawlee` + `Playwright`. Это позволяет парсить современные SPA-сайты (React/Vue), которые не под силу обычным библиотекам типа `requests`.
- **Обход:** Автоматический поиск внутренних ссылок и рекурсивный обход в пределах одного домена.
- **Ограничения:** Установлен лимит на количество запросов (10 страниц) и таймауты, чтобы избежать бесконечного цикла.
- **Чистка данных:** Используются регулярные выражения и фильтрация дубликатов через `set`.

## Требования
- Python 3.12+
- Node.js (для работы Playwright браузеров)

## Установка

1. **Создайте виртуальное окружение и активируйте его:**
   ```
   python -m venv venv
   source venv/Scripts/activate  # Для Windows
   source venv/bin/activate      # Для Linux/macOS
   ```
2. **Установите основные зависимости:**
   ```
   pip install -r requirements.txt
   ```
3. **Установите браузер Chromium:**
   ```
   playwright install chromium
   ```

## Запуск
1. **Запустите основной скрипт из корня проекта:**
   ```
   python parser.py
   ```
2. **В терминале появится приглашение к вводу:**
   ```
   Введите адрес сайта:
   ```
3. **Введите полный URL (обязательно с протоколом http:// или https://) и нажмите Enter:**
   ```
   https://example.com
   ```

## Результат
По завершении работы программа выведет итоговый объект в консоль.
   ```
   {
    "url": "https://example.com",
    "emails": ["info@example.com"],
    "phones": ["79001234567"]
	}
	```
